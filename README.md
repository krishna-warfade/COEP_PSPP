# COEP_PSPP

**Problem Solving Using Procedural Programming (PSPP)**

This repository contains a collection of Assignments, Lab Tasks, and End-Semester Examination questions for the "Problem Solving Using Procedural Programming" course (Semester 1) at COEP.

The code is written in **C**.
